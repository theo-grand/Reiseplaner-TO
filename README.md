# Travel Planner Application

## Overview
This is a travel planner application designed to help users organize their trips efficiently. Users can plan itineraries, manage bookings, and keep track of important travel information.

## Features
- **Itinerary Creation**: Users can create detailed itineraries for each trip, including transportation, accommodation, and activities.
- **Booking Management**: Integration with booking platforms to manage flights, hotels, and car rentals.
- **Travel Alerts**: Notifications for flight changes, weather updates, and local events.
- **Sharing**: Users can share their itineraries with friends and family or collaborate on trip planning.

## Technologies Used
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Database: MongoDB

## Installation
To install the application, clone the repository and install the dependencies:
```bash
git clone https://github.com/theo-grand/Reiseplaner-TO.git
cd Reiseplaner-TO
npm install
```

## Usage
Run the application using:
```bash
npm start
```
Visit `http://localhost:3000` in your browser to access the application.
